# maxNumberOfSpans

`val maxNumberOfSpans: `[`Int`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-int/index.html)`?`

Maximum number of spans. Default = null, meaning no limit. For example, if value = 3, the annotation can have
3 separate spans.
